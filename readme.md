# E-Commerce Database & Analytics Project

## **Project Overview**
This project demonstrates a **complete e-commerce database workflow** along with advanced analytics. It covers:

- **Database Creation & Management** – designing normalized schemas, constraints, and relationships.
- **Sample Data Generation** – simulating realistic data for customers, products, orders, and payments.
- **Analytics & Business Insights** – sales, customer behavior, product performance, payment trends, and advanced analyses like cohort analysis, RFM segmentation, and market basket analysis.

**Project Scope:**
- 1,000+ Customers
- 5,000+ Products with 3 variants each
- 1,500+ Orders with multiple Order Items
- Payments, Discounts, and Returns incorporated
- Designed to showcase **PostgreSQL skills, SQL analytics, and business intelligence insights**.
